```javascript
nextBigger(12)==21
nextBigger(513)==531
nextBigger(2017)==2071
```
```python
next_bigger(12)==21
next_bigger(513)==531
next_bigger(2017)==2071
```
```ruby
next_bigger(12)==21
next_bigger(513)==531
next_bigger(2017)==2071
```
```csharp
Kata.NextBiggerNumber(12)==21
Kata.NextBiggerNumber(513)==531
Kata.NextBiggerNumber(2017)==2071
```
```java
Kata.nextBiggerNumber(12)==21
Kata.nextBiggerNumber(513)==531
Kata.nextBiggerNumber(2017)==2071
```
